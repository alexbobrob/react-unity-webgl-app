# HTML Structure

The structure of the HTML should be as follows:

```html
<!DOCTYPE html>
<html lang="nl">
  <head>
    <title>My Unity Game</title>
  </head>
  <body>
    <div id="app"></div>
  </body>
  <script src="Build/UnityLoader.js"></script>
  <script src="compiled/bundle.js"></script>
</html>
```
